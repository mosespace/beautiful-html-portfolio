# Beautiful Portfolio

A personal portfolio website template built with HTML and CSS.

## Current Features

- Static HTML structure with portfolio sections
- CSS styling with custom color scheme
- About section with personal details
- Skills showcase (4 skills: JavaScript, React, Node.js, MongoDB)
- Resume section (education & experience)
- Contact form in footer
- Social media icon links

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Open Sans)
- FontAwesome 5 Icons

## Main Sections

- **Header/Navigation** - Logo and navigation links
- **Banner** - Hero section with name and intro
- **About** - Personal information and background
- **Skills** - Technology stack showcase
- **Resume** - Education and work experience
- **Footer** - Contact form and social links
